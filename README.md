# ZOTCrawler
A UCI domain-based crawler that traverses through 4 initial seeds and a search engine that has over 700k documents!

## Requirements
- Python 3.12
- Flask
- NLTK tokenizer

## Quick start 
- python searchWebGui.py : web interface
- python main.py : terminal interface
